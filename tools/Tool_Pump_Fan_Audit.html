🌲 Fault Tree — คลิกเพื่อดูสาเหตุ & วิธีแก้
🌀 CAVITATION (Pump)
🌊 Cavitation — NPSH_a < NPSH_r + Safety Margin
เสียงดังคล้ายกรวด (Rattling) · Vibration สูงผิดปกติ · Head ตก · Impeller กร่อน
สาเหตุ:
• P_suction ต่ำ — ท่อดูดยาว, Filter/Strainer อุดตัน
• T_fluid สูง → P_vapor สูง → NPSH_a ลด
• ปั้มอยู่สูงกว่าถังน้ำมากเกินไป (Z_s สูง)
• Flow เกิน Runout → ปั้มทำงานนอก Curve
แนวทางแก้ไข: ✓ ทำความสะอาด Filter/Strainer บนท่อดูด
✓ ลดความสูงติดตั้งปั้ม (ถ้าทำได้)
✓ เพิ่ม Back Pressure ด้าน Suction (ถ้าระบบปิด)
✓ ลด Flow Rate ด้วย Discharge Valve
✓ ตรวจ T_fluid — ลดอุณหภูมิของเหลวถ้าเป็นไปได้
⬇️ LOW EFFICIENCY
📉 Efficiency ต่ำกว่า Design >5%
η_actual < η_design − 5% → ซ่อมบำรุง, Impeller สึก, Wear Ring กว้าง
สาเหตุ:
• Wear Ring Gap กว้างขึ้น → Internal Recirculation (Pump)
• Blade Fouling / Erosion — ตะกอน, กัดกร่อน (Fan)
• Impeller Worn หรือ Damaged
• Belt Slip → P_shaft น้อยกว่าที่คาด (Fan Belt Drive)
แนวทางแก้ไข: ✓ วัด Wear Ring Clearance — เกิน 2× Original → เปลี่ยน
✓ ทำความสะอาด Impeller/Blade ทุก 3–6 เดือน
✓ ตรวจ Belt Tension — D = L/100 cm
✓ ถ้า η < 60% ของ Design พิจารณา Overhaul/Replace
🔄 Operating Point ห่างจาก BEP >20%
Flow น้อยหรือมากกว่า BEP มาก → Efficiency ต่ำ, Vibration สูง, อายุสั้น
ผลกระทบ:
• Q < 0.7×Q_BEP: Recirculation Noise, Impeller Load สูง
• Q > 1.2×Q_BEP: NPSH Problems, Overload Motor
แนวทางแก้ไข: ✓ ติด VFD ปรับ Speed ให้ Operating Point ใกล้ BEP
✓ เปลี่ยน Impeller Diameter (Trimming) — ลด Head
✓ ติด Bypass Line (ถ้า Q ต่ำกว่า BEP มาก)
🌀 FAN SURGE / STALL (Fan Only)
🔀 Fan Surge / Stall — Unstable Operation
Pressure ขึ้นลงไม่สม่ำเสมอ · เสียงดัง Pulsating · Vibration สูง · เกิดในบริเวณ S-Curve ของ H-Q
สาเหตุ:
• ระบบ Ductwork มี Static Resistance สูงเกินไป
• Damper ปิดมากเกินไป → Q ต่ำกว่า Surge Point
• Fan ทำงานซ้อน (Parallel) ที่ Resistance สูง
แนวทางแก้ไข: ✓ ลด System Resistance — ขยายขนาดท่อ, ลด Damper
✓ ปรับ Inlet Vane ถ้ามีระบบ Inlet Guide Vane
✓ ใช้ VFD ปรับ Speed แทน Damper Control
✓ Anti-Surge Bypass Valve ในระบบ Critical
📳 VIBRATION & MECHANICAL
📳 Vibration สูงเกิน ISO 10816 Limit
Bearing เสียหาย, Imbalance, Misalignment, Cavitation
วิเคราะห์จาก Vibration Spectrum:
• 1× RPM → Rotor Unbalance → Dynamic Balancing
• 2× RPM → Misalignment → Laser Alignment
• สูง Frequency (BPFO/BPFI) → Bearing Defect → เปลี่ยน Bearing
• Broadband Noise → Turbulence, Cavitation
• 2× Line Freq (100Hz) → Electrical (Motor Eccentric)
แนวทางแก้ไข: ✓ Dynamic Balancing In-Situ (ถ้า >1×RPM dominant)
✓ Laser Alignment ระหว่าง Motor–Pump/Fan
✓ เปลี่ยน Bearing (SPM Shock Pulse Method)
✓ ตรวจ Foundation Bolt Torque
